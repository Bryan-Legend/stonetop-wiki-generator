# Category icons

White-on-transparent SVGs from [game-icons.net](https://game-icons.net/)
(CC BY 3.0 — credit Lorc, Delapouite, Carl Olsen, and other authors as noted
on each icon’s page).

CSS (`.book-icon`) recolors the white fills to the wiki accent (`--accent`).

Do not commit PDF-extracted bitmaps here; those are replaced by these assets.
